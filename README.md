# AutoMyBiz Shop Themes

Custom CSS themes for GoHighLevel (GHL) Shops to override default styling and target specific elements like text colors.

## Usage

To use these themes in your GHL Shop:

1. Find the CSS file you want to use (e.g., `dark-teal.css`).
2. Copy the raw URL from GitHub or use the GitHub Pages link:
   `https://automybiz.github.io/AutoMyBiz-Shop-Themes/dark-teal.css`
3. Add the link to your GHL website's header or custom CSS section.

## Themes

- `dark-teal.css`: A dark theme with teal accents.
